Documentação Completa dos Parâmetros do CFW500 (V3.8X e Modbus RTU)

Este documento detalha os parâmetros do inversor de frequência WEG CFW500, combinando informações do Manual de Programação V3.8X e do Manual do Usuário Modbus RTU.

Notas Gerais:

Acesso via Modbus: A maioria dos parâmetros é acessível via Modbus RTU como Holding Registers (16 bits), onde o endereço Modbus geralmente corresponde ao número do parâmetro (Ex: P0308 é o Holding Register 308).

Propriedades:

ro: Somente Leitura (Read Only).

cfg: Configuração (geralmente alterável com motor parado).

sy: Parâmetro relacionado à Segurança Funcional (STO/SS1-t).

V/f, VVW, VVW PM, Vetorial, Sless, Enc: Indica os modos de controle do motor em que o parâmetro é visível/aplicável.

-: Parâmetro de escrita/leitura ou comando.

Grupos HMI: Indica o grupo de menu na Interface Homem-Máquina.

Padrão: Valor de fábrica. Valores entre parênteses ( ) geralmente indicam o padrão para 50 Hz.

Parâmetros Principais e de Leitura (Grupo HMI: READ, BASIC, etc.)

P0000 – Acesso aos Parâmetros

Faixa de Valores: 0 a 9999

Padrão: 0

Propriedades: -

Grupos de Acesso via HMI: -

Descrição: Usado para inserir a senha (definida em P0200) e liberar o acesso à alteração de parâmetros. Valor 1 libera, 0 bloqueia (após ajuste, volta a indicar 1 ou 0).

P0001 – Referência Velocidade

Faixa de Valores: 0 a 65535

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Valor da referência de velocidade atual na unidade/escala definida por P0208-P0213.

P0002 – Velocidade de Saída (Motor)

Faixa de Valores: 0 a 65535

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica a velocidade imposta na saída do inversor (antes de compensações) na mesma escala/unidade de P0001.

P0003 – Corrente do Motor

Faixa de Valores: 0,0 a 400,0 A

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica a corrente de saída total RMS do inversor em Amperes.

P0004 – Tensão Link DC (Ud)

Faixa de Valores: 0 a 2000 V

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica a tensão no barramento de corrente contínua (Link DC) em Volts.

P0005 – Frequência de Saída (Motor)

Faixa de Valores: 0,0 a 500,0 Hz

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica a frequência real instantânea aplicada ao motor em Hertz (Hz), incluindo compensações.

P0006 – Estado do Inversor

Faixa de Valores: 0=Ready, 1=Run, 2=Subtensão, 3=Falha, 4=Autoajuste, 5=Configuração, 6=Frenagem CC, 7=STO, 8=Fire Mode, 9=Reservado, 10=Modo dormir

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica o estado atual de operação do inversor.

P0007 – Tensão de Saída

Faixa de Valores: 0 a 2000 V

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica a tensão de linha na saída do inversor (referência ou estimada) em Volts.

P0009 – Torque no Motor

Faixa de Valores: -1000,0 a 1000,0 %

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica o torque desenvolvido pelo motor em relação ao torque nominal (calculado).

P0010 – Potência de Saída

Faixa de Valores: 0,0 a 6553,5 kW

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica a potência elétrica na saída do inversor (calculada).

P0011 – Fator de Potência

Faixa de Valores: -1,00 a 1,00

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica o fator de potência (relação entre potência ativa e aparente) absorvido pelo motor.

P0012 – Estado das Entradas Digitais DI8 a DI1

Faixa de Valores: Bitmap (00h a FFh)

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ, I/O

Descrição: Indica o estado (0=Inativa, 1=Ativa) de cada entrada digital (DI1 no Bit 0 até DI8 no Bit 7).

P0013 – Estado das Saídas Digitais DO5 a DO1

Faixa de Valores: Bitmap (00h a 1Fh)

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ, I/O

Descrição: Indica o estado (0=Inativa/Aberta, 1=Ativa/Fechada) de cada saída digital (DO1 no Bit 0 até DO5 no Bit 4).

P0014/P0015 – Valor da Saída Analógica AO1/AO2

Faixa de Valores: 0,0 a 100,0 %

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ, I/O

Descrição: Indicam o valor percentual atual das saídas analógicas AO1 e AO2, após ganho.

P0016/P0017 – Valor da Saída em Frequência FO em % / Hz

Faixa de Valores: 0,0 a 100,0 % / 0 a 20000 Hz

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ, I/O

Descrição: Indicam o valor percentual (P0016) ou em Hertz (P0017) da saída em frequência (FO).

P0018/P0019/P0020 – Valor da Entrada Analógica AI1/AI2/AI3

Faixa de Valores: -100,0 a 100,0 %

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ, I/O

Descrição: Indicam o valor percentual das entradas analógicas AI1, AI2 e AI3, após offset e ganho.

P0021/P0022 – Valor da Entrada em Frequência FI em % / Hz

Faixa de Valores: -100,0 a 100,0 % / 0 a 20000 Hz

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ, I/O

Descrição: Indicam o valor percentual (P0021) ou em Hertz (P0022) da entrada em frequência (FI), após offset e ganho.

P0023/P0024 – Versão de Software Principal/Secundário

Faixa de Valores: 0,00 a 655,35

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indicam as versões do firmware principal (cartão de controle) e secundário (módulo plug-in).

P0027 – Configuração do Módulo Plug-in

Faixa de Valores: 0 a 17 (ver Tabela 6.1 no manual)

Padrão: - (Depende do módulo)

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Identifica o tipo de módulo plug-in conectado.

P0028 – Módulo de Funções de Segurança

Faixa de Valores: 0=Jumper STO, 1=CFW500-SFY2

Padrão: - (Depende do módulo)

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Identifica se o módulo de segurança SFY2 está instalado.

P0029 – Configuração do Hardware de Potência

Faixa de Valores: 0 a 60 (Ver lista no manual)

Padrão: Conforme modelo

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Identifica o modelo de hardware de potência (tensão/corrente) do inversor.

P0030 – Temperatura do Módulo de Potência

Faixa de Valores: -20 a 150 ºC

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Temperatura medida no dissipador do módulo de potência via NTC.

P0034 – Temperatura do Ar Interno

Faixa de Valores: -20 a 150 ºC

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Temperatura interna do ar dentro do inversor.

P0036 – Velocidade do Ventilador Int.

Faixa de Valores: 0 a 15000 rpm

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Velocidade atual do ventilador interno do dissipador.

P0037 – Sobrecarga do Motor Ixt

Faixa de Valores: 0 a 100 %

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Nível percentual atual do integrador Ixt da proteção de sobrecarga térmica do motor. 100% = falha F0072.

P0038 – Velocidade do Encoder

Faixa de Valores: 0 a 65535 rpm

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Velocidade lida do encoder (se modo vetorial com encoder ativo).

P0039 – Contador dos Pulsos do Encoder

Faixa de Valores: 0 a 40000

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Contagem de pulsos do encoder (se modo vetorial com encoder ativo).

P0040 – Variável de Processo PID

Faixa de Valores: 0,0 a 3000,0

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Valor atual da variável de processo (realimentação) do regulador PID, na escala P0528/P0529.

P0041 – Valor do Setpoint PID

Faixa de Valores: 0,0 a 3000,0

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Valor atual do setpoint (referência) do regulador PID, na escala P0528/P0529.

P0042 – Tempo Energizado

Faixa de Valores: 0 a 65535 h

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Tempo total em que o inversor permaneceu energizado (não reseta com P0204=3).

P0043 – Tempo Ativado

Faixa de Valores: 0,0 a 6553,5 h

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Tempo total em que o inversor esteve habilitado (motor rodando ou em frenagem CC). Reseta com P0204=3.

P0044 – Contador de kWh

Faixa de Valores: 0 a 65535 kWh

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Energia consumida pelo motor (calculada, não para medição fiscal). Reseta com P0204=4.

P0047 – Estado CONFIG

Faixa de Valores: 0 a 999

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica o motivo específico pelo qual o inversor entrou no estado de Configuração (parâmetros incompatíveis). Ver Tabela 0.1.

P0048 – Alarme Atual

Faixa de Valores: 0 a 999

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Indica o número do alarme ativo no momento. 0 = Sem alarme.

P0049/P0050/P0060/P0070 – Falha Atual / Última / Segunda / Terceira Falha

Faixa de Valores: 0 a 999

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Histórico das 3 últimas falhas ocorridas e a falha atual (se houver).

P0051/P0061/P0071 – Corrente de Saída Última / Segunda / Terceira Falha

Faixa de Valores: 0,0 a 400,0 A

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Corrente de saída no instante das 3 últimas falhas.

P0052/P0062/P0072 – Link DC Última / Segunda / Terceira Falha

Faixa de Valores: 0 a 2000 V

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Tensão do Link DC no instante das 3 últimas falhas.

P0053/P0063/P0073 – Frequência de Saída Última / Segunda / Terceira Falha

Faixa de Valores: 0,0 a 500,0 Hz

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Frequência de saída no instante das 3 últimas falhas.

P0054/P0064/P0074 – Temperatura nos IGBTs Última / Segunda / Terceira Falha

Faixa de Valores: -20 a 150 ºC

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Temperatura do módulo de potência no instante das 3 últimas falhas.

P0055/P0065/P0075 – Estado Lógico Última / Segunda / Terceira Falha

Faixa de Valores: 0000h a FFFFh

Padrão: -

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Estado lógico (P0680) no instante das 3 últimas falhas.

P0080/P0081/P0082 – Última / Segunda / Terceira Falha em "Fire Mode"

Faixa de Valores: 0 a 999

Padrão: 0

Propriedades: ro

Grupos de Acesso via HMI: READ

Descrição: Histórico das 3 últimas falhas ocorridas durante a operação em "Fire Mode".

Parâmetros de Rampas (Grupo HMI: BASIC, I/O)

P0100/P0101 – Tempo Aceleração/Desaceleração 1

Faixa de Valores: 0,1 a 999,0 s

Padrão: 10,0 s (Alterado para 20,0 s nas mecânicas F/G)

Propriedades: -

Grupos de Acesso via HMI: BASIC

Descrição: Define os tempos da 1ª rampa (padrão) para ir de 0 a P0134 (acel) e de P0134 a 0 (desacel).

P0102/P0103 – Tempo Aceleração/Desaceleração 2ª Rampa

Faixa de Valores: 0,1 a 999,0 s

Padrão: 10,0 s (Alterado para 20,0 s nas mecânicas F/G)

Propriedades: -

Grupos de Acesso via HMI: -

Descrição: Define os tempos da 2ª rampa, ativada via P0105.

P0104 – Rampa S

Faixa de Valores: 0=Inativa, 1=Ativa

Padrão: 0

Propriedades: cfg

Grupos de Acesso via HMI: -

Descrição: Habilita/desabilita o perfil de rampa em "S" para suavizar acelerações/desacelerações.

P0105 – Seleção 1ª/2ª Rampa

Faixa de Valores: 0=1ª Rampa, 1=2ª Rampa, 2=DIx, 3=Serial/USB, 4=Reservado, 5=CO/DN/PB/Eth, 6=SoftPLC

Padrão: 2

Propriedades: -

Grupos de Acesso via HMI: I/O

Descrição: Define a fonte que selecionará entre a 1ª e a 2ª rampa.

P0106 – Tempo da 3ª Rampa

Faixa de Valores: 0,1 a 999,0 s

Padrão: 5,0 s

Propriedades: -

Grupos de Acesso via HMI: -

Descrição: Tempo de aceleração/desaceleração usado por funções de proteção (Limitação de Corrente, Regulação Link DC, Parada Rápida, SS1-t).

Parâmetros de Segurança Funcional

P0108 – Tempo SS1-t

Faixa de Valores: 0 a 999 s

Padrão: -

Propriedades: sy

Grupos de Acesso via HMI: -

Descrição: Define o tempo de atraso para a função Safe Stop 1 (SS1-t) no módulo de segurança.

P0109 – Confirmação Tempo SS1-t

Faixa de Valores: 0 a 999 s

Padrão: -

Propriedades: ro, sy

Grupos de Acesso via HMI: -

Descrição: Mostra o tempo SS1-t real programado no módulo de segurança para confirmação.

Parâmetros de Referência de Velocidade (Grupo HMI: BASIC, I/O, etc.)

P0120 – Backup da Ref. Veloc.

Faixa de Valores: 0=Inativo, 1=Ativo, 2=Backup por P0121

Padrão: 1

Propriedades: -

Grupos de Acesso via HMI: -

Descrição: Define se a última referência de velocidade é salva ao desabilitar ou desligar (Ativo), se volta para P0133 (Inativo), ou se sempre parte de P0121 (Backup por P0121).

P0121 – Referência via HMI

Faixa de Valores: 0,0 a 500,0 Hz

Padrão: 3,0 Hz

Propriedades: -

Grupos de Acesso via HMI: -

Descrição: Armazena o valor da referência ajustada pelas teclas da HMI.

P0122 – Referência JOG

Faixa de Valores: -500,0 a 500,0 Hz

Padrão: 5,0 Hz

Propriedades: -

Grupos de Acesso via HMI: -

Descrição: Define a velocidade para a função JOG.

P0124 a P0131 – Referência 1 a 8 Multispeed

Faixa de Valores: -500,0 a 500,0 Hz

Padrões: 3.0, 10.0(5.0), 20.0(10.0), 30.0(20.0), 40.0(30.0), 50.0(40.0), 60.0(50.0), 66.0(55.0) Hz

Propriedades: -

Grupos de Acesso via HMI: -

Descrição: Define os 8 níveis de velocidade para a função Multispeed.

P0132 – Nivel Máx. Sobreveloc

Faixa de Valores: 0 a 100 %

Padrão: 10 %

Propriedades: cfg

Grupos de Acesso via HMI: BASIC

Descrição: Percentual acima de P0134 que, se excedido por 20ms, causa falha F0150. 100% desabilita.

P0133 – Velocidade Mínima

Faixa de Valores: 0,0 a 500,0 Hz

Padrão: 3,0 Hz

Propriedades: -

Grupos de Acesso via HMI: BASIC

Descrição: Limite mínimo para a referência de velocidade.

P0134 – Velocidade Máxima

Faixa de Valores: 0,0 a 500,0 Hz

Padrão: 66,0 (55,0) Hz

Propriedades: -

Grupos de Acesso via HMI: BASIC

Descrição: Limite máximo para a referência de velocidade. Base para cálculo das rampas.


Parâmetros de Controle do Motor (Grupo HMI: BASIC, MOTOR, etc.)

P0135 – Corrente Máxima Saída

Faixa de Valores: 0,0 a 400,0 A

Padrão: 1,5x Inom

Propriedades: V/f, VVW, VVW PM

Grupos de Acesso via HMI: BASIC, MOTOR

Descrição: Nível de corrente para ativar a limitação de corrente (Modos Hold de Rampa ou Desacelera Rampa em P0150).

P0136 – Boost de Torque Manual

Faixa de Valores: 0,0 a 30,0 %

Padrão: Conforme modelo

Propriedades: V/f, VVW PM

Grupos de Acesso via HMI: BASIC, MOTOR

Descrição: Aumenta a tensão de saída em baixas frequências (0 a P0147) para compensar queda de tensão e manter torque.

P0137 – Boost de Torque Autom.

Faixa de Valores: 0,0 a 30,0 %

Padrão: 0,0 %

Propriedades: V/f

Grupos de Acesso via HMI: MOTOR

Descrição: Boost de torque proporcional à corrente ativa (similar ao P0136).

P0138 – Compensação Escorreg.

Faixa de Valores: -10,0 a 10,0 %

Padrão: 0,0 %

Propriedades: V/f

Grupos de Acesso via HMI: MOTOR

Descrição: Compensa o escorregamento do motor, aumentando a frequência de saída com o aumento da corrente ativa, para manter velocidade constante com carga.

P0139 – Filtro Corrente Saída

Faixa de Valores: 0 a 9999 ms

Padrão: 50 ms

Propriedades: V/f, VVW

Grupos de Acesso via HMI: -

Descrição: Constante de tempo do filtro para a corrente de saída total e ativa.

P0140 – Filtro Comp. Escorreg.

Faixa de Valores: 0 a 9999 ms

Padrão: 500 ms

Propriedades: VVW

Grupos de Acesso via HMI: -

Descrição: Constante de tempo do filtro para a compensação de escorregamento (usada no controle VVW).

P0142/P0143/P0144 – Tensão Saída Máxima / Intermediária / Mínima

Faixa de Valores: 0,0 a 100,0 %

Padrões: 100,0%, 66,7%, 33,3%

Propriedades: cfg, V/f, VVW PM

Grupos de Acesso via HMI: -

Descrição: Definem os níveis de tensão (% da tensão nominal) nos pontos P3, P2 e P1 da curva V/f, respectivamente.

P0145/P0146/P0147 – Freq. Início Enf. Campo / Saída Intermed. / Saída Baixa

Faixa de Valores: 0,0 a 500,0 Hz

Padrões: 60(50) Hz, 40(33.3) Hz, 20(16.7) Hz

Propriedades: cfg, V/f, VVW PM

Grupos de Acesso via HMI: -

Descrição: Definem as frequências correspondentes aos pontos P3, P2 e P1 da curva V/f, respectivamente. P0145 é a frequência nominal.

P0148 – Ação do V/f

Faixa de Valores: 0=V/f padrão, 1=Soft-Starter (tensão)

Padrão: 0

Propriedades: cfg, V/f, VVW PM

Grupos de Acesso via HMI: -

Descrição: 0: Controle V/f normal. 1: Soft-Starter - varia apenas tensão, frequência fixa (P0001).

P0150 – Tipo Regul. Link DC V/f

Faixa de Valores: 0=Hold_Ud/Desac_LC, 1=Acel_Ud/Desac_LC, 2=Hold_Ud/Hold_LC, 3=Acel_Ud/Hold_LC

Padrão: 0

Propriedades: cfg, V/f, VVW

Grupos de Acesso via HMI: MOTOR

Descrição: Configura como a rampa reage à limitação de tensão do Link DC (Ud) e Limitação de Corrente (LC). (Hold=congela, Acel=acelera, Desac=desacelera).

P0151 – Nível Regul. Ud V/f

Faixa de Valores: 339 a 1200 V

Padrão: Depende de P0296 (Ver Tabela 9.1)

Propriedades: V/f, VVW, VVW PM

Grupos de Acesso via HMI: -

Descrição: Nível de tensão do Link DC para ativar a regulação (conforme P0150).

P0152 – Ganho Prop. Regul. Ud V/f

Faixa de Valores: 0,00 a 9,99

Padrão: 1,50

Propriedades: V/f, VVW, VVW PM

Grupos de Acesso via HMI: -

Descrição: Ganho proporcional para o modo "Acelera Rampa" (P0150=1 ou 3) da regulação do Link DC.

P0153 – Nível Frenagem Reost.

Faixa de Valores: 339 a 1200 V

Padrão: Depende de P0296 (Ex: 375V para 200-240Vca)

Propriedades: -

Grupos de Acesso via HMI: -

Descrição: Nível de tensão do Link DC para ativar o IGBT de frenagem (requer resistor externo).

Padrões: 1,1xInom, 1,0xInom, 0,8xInom

Grupos de Acesso via HMI: -

APPEND: Análise da Configuração das Tabelas 1 e 2

(Conforme análise anterior)

Tabela 1: Configurações Gerais do CFW500 e do Motor

Configura o inversor para um motor específico (0.25 kW, 220V, 1.5A, 60Hz, 1700rpm) usando controle V/f.

Define rampas de 5s com perfil S.

Ajusta limites de velocidade e corrente (150% In).

Desativa compensação de escorregamento e Boost Automático. Boost Manual em 5.0% (possível inconsistência com justificativa, verificar necessidade).

Configura tensão e frequência de chaveamento.

Tabela 2: Configurações para Comunicação via Modbus RTU no CFW500

Configura o inversor para operar em modo Remoto, recebendo comandos e referência via Serial (Modbus RTU).

Define parâmetros Modbus: Endereço 1, Velocidade 19200 bps, Formato 8N1.

Habilita protocolo Modbus RTU na interface padrão (Serial 1).

Configura ação de segurança: Parar por rampa se ocorrer timeout de comunicação (watchdog) de 5 segundos.

Indica quais parâmetros (P0682, P0683) serão usados pelo mestre para controle.

Conclusão do APPEND: A configuração combinada das tabelas prepara o CFW500 para controlar o motor especificado usando controle V/f, recebendo todos os comandos e setpoints via Modbus RTU (endereço 1, 19200, 8N1), com parada segura em caso de falha de comunicação superior a 5 segundos.

Análise da Configuração da Planilha de Resumo ("Tabela 1" e "Tabela 2")

Com base nos valores e justificativas presentes nas tabelas fornecidas na última página do OCR:

Tabela 1: Configurações Gerais do CFW500 e do Motor

P0100 (Tempo Acel 1): 5,0 s - Definido conforme cálculo prévio.

P0101 (Tempo Desacel 1): 5,0 s - Mantido igual à aceleração para simetria.

P0104 (Rampa S): 1 (Curva S) - Ativada para suavizar partidas/paradas e reduzir choques mecânicos.

P0105 (Seleção 1ª/2ª Rampa): 0 (1ª Rampa) - Utilizará sempre a rampa principal, sem necessidade de alternar.

P0133 (Velocidade Mínima): 0,0 Hz - Permite partir de velocidade zero.

P0134 (Velocidade Máxima): 60,0 Hz - Definida para a frequência nominal do motor.

P0135 (Corrente Máx Saída): 2,2 A - Ajustado para 150% da corrente nominal do motor (1,5 A), provendo proteção e permitindo sobrecorrentes momentâneas.

P0136 (Boost Torque Manual): 5,0 % - Desativado (Nota: A descrição diz desativado, mas o valor é 5.0%. A justificativa indica que não é necessário aumento de torque em baixas velocidades, o que implica em um valor baixo ou zero. Pode haver uma inconsistência ou o 5.0% é considerado desativado na prática para esta aplicação).

P0137 (Boost Torque Automático): 0,0 % - Mantido em 0%, sem compensação adicional de torque.

P0138 (Compensação Escorreg.): 0,0 % - Desativada, pois o escorregamento é mínimo em velocidades nominais.

P0202 (Tipo de Controle): 0 (V/f) - Controle escalar V/f, adequado para operação próxima à velocidade nominal, sem necessidade de controle vetorial.

P0296 (Tensão Nominal Rede): 0 (200-240 V) - Configurado conforme tensão de alimentação disponível (220V).

P0297 (Frequência Chaveamento): 7500 Hz - Valor ajustado conforme necessidade da aplicação (implica em menor ruído acústico).

P0400 (Tensão Nominal Motor): 220 V - Conforme especificação do motor.

P0401 (Corrente Nominal Motor): 1,5 A - Calculada baseada na potência (245W) e tensão nominal do motor.

P0402 (Velocidade Nominal Motor): 1700 rpm - Conforme memorial de cálculo (especificação do motor).

P0403 (Frequência Nominal Motor): 60,0 Hz - Frequência padrão da rede e correspondente à velocidade nominal.

P0404 (Potência Nominal Motor): 2 (0,25 kW / 0,33 HP) - Código selecionado mais próximo da potência real do motor (245W ≈ 0,25 kW).

P0407 (Fator Potência Motor): 0,65 - Valor típico para motores de baixa potência, conforme especificado.

Tabela 2: Configurações para Comunicação via Modbus RTU no CFW500

P0220 (Fonte Local/Remoto): 1 (Remoto) - Configura o inversor para ser controlado via modo Remoto (necessário para controle via Modbus).

P0222 (Ref. Remota): 9 (Serial) - Define que a referência de velocidade no modo Remoto virá da interface Serial (Modbus).

P0226 (Giro Remoto): 5 (Serial) - Permite que o comando de sentido de giro no modo Remoto seja via interface Serial (Modbus).

P0227 (Gira/Para Remoto): 2 (Serial) - Define que o comando Gira/Para no modo Remoto virá da interface Serial (Modbus).

P0308 (Endereço Serial): 1 - Define o endereço 1 para o escravo Modbus RTU (deve ser único na rede).

P0310 (Taxa Comunicação): 1 (19200 bps) - Define a velocidade da comunicação Modbus. Deve coincidir com a do mestre (ex: Raspberry Pi Pico).

P0311 (Config. Bytes): 0 (8N1) - Configura o formato 8 bits de dados, sem paridade, 1 stop bit. Deve coincidir com o do mestre.

P0312 (Protocolo Serial): 2 (Modbus RTU1) - Seleciona Modbus RTU na interface serial padrão (bornes RS485), permitindo controle via RS485.

P0313 (Ação Erro Comunic.): 1 (Para por Rampa) - Em caso de erro de comunicação, o motor para suavemente usando a rampa de desaceleração.

P0314 (Watchdog Serial): 5,0 s - Define um timeout de 5 segundos. Se não receber mensagens válidas nesse tempo, executa a ação de P0313.

P0316 (Estado Interf. Serial): - (Verificado) - Indica que se deve verificar se a interface está ativa (valor = 1).

P0682 (Palavra Controle Serial): - (Manipulada) - Será utilizada para enviar comandos (Habilita, Gira/Para) via Modbus pelo controlador externo.

P0683 (Ref. Velocidade Serial): - (Manipulada) - Será utilizada para enviar a referência de velocidade via Modbus pelo controlador externo.

Conclusão da Análise da Configuração:

A configuração detalhada nas Tabelas 1 e 2 visa preparar o inversor CFW500 para:

Controlar um motor de baixa potência (~0.25 kW, 220V, 1.5A, 60Hz) usando o modo de controle escalar V/f.

Operar primariamente em modo Remoto.

Receber os comandos principais (Gira/Para, Sentido de Giro) e a referência de velocidade através da interface serial padrão (RS485) utilizando o protocolo Modbus RTU.

Comunicar-se a 19200 bps com formato de dados 8N1 e endereço de escravo 1.

Implementar uma medida de segurança: em caso de perda de comunicação Modbus por mais de 5 segundos, o inversor deve parar o motor utilizando a rampa de desaceleração configurada.

Utilizar rampas de aceleração/desaceleração de 5 segundos com perfil em "Curva S" para suavidade.

A proteção de sobrecarga do motor está ajustada para 150% da corrente nominal.
